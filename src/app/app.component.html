Hellblaze est un prototype de jeu RPG / Beat'em'all à la 3ème personne. Il a été réalisé lors de notre formation en création de jeux vidéo de l'Université du Québec à Chicoutimi. Vous incarnez un anti-héros dont le but est de regagner son statut de démon. Il doit pour cela progresser en accomplissant différents objectifs, comme par exemple tuer un certain nombre de monstres ou bien terrasser un boss.

Contrôles
  • WASD / Joystick gauche : bouger le personnage
  • Souris / Joystick droit : bouger la caméra
  • Clic gauche / X : attaquer
  • Echap / Start : pause
  • I / Select : stats de progression

À l'occasion de la Global Game Jam 2020, nous avons formé une équipe de 8 passionnés pour réaliser un jeu en 48h sur le thème "Repair". Le principe est simple : vous êtes un robot chargé de protéger les derniers "datacenters" d'une humanité disparue. Une IA a trouvé un moyen de hacker ces serveurs et veut les détruire. Vous devez alors courir de serveur en serveur pour les réparer avant le temps imparti pour protéger les données.

Contrôles
  • WASD / Joystick gauche : bouger le personnage
  • Entrée / A : Réparer

Projet en cours : 2D Fight

Infos

Un "Versus Fighting Game" en 2D qui aura pour but de rassembler (ou fâcher) les joueurs.

Contrôles
  • AD / Joystick gauche : bouger le personnage
  • Espace / A : sauter
  • Clic gauche / X : tirer
Conseils
  • Utilisez les murs pour continuer vos sauts !
  • Gérer vos flêches, elles sont limitées !
  • Ramassez des flêches quand vous en êtes à cours !
