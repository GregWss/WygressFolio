Hellblaze

Infos

Hellblaze est un prototype de jeu RPG / Beat'em'all à la 3ème personne. Il a été réalisé lors de notre formation en création de jeux vidéo de l'Université du Québec à Chicoutimi.

Contrôles
  • WASD / Joystick gauche : bouger le personnage
  • Souris / Joystick droit : bouger la caméra
  • Clic gauche / X : attaquer
  • Echap / Start : pause
  • I / Select : stats de progression

Projet en cours : 2D Fight

Infos

Un "Versus Fighting Game" en 2D qui a pour but de rassembler (ou fâcher) les joueurs.

Contrôles
  • AD / Joystick gauche : bouger le personnage
  • Espace / A : sauter
  • Clic gauche / X : tirer
Conseils
  • Utilisez les murs pour continuer vos sauts !
  • Gérer vos flêches, elles sont limitées !
  • Ramassez des flêches quand vous en êtes à cours !

Vous pouvez voir le code source du jeu sur sa page GitHub !
